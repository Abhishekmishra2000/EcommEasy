# E-commAPI - An Ecommerce API
E-commAPI is Node.js based eCommerce API. 

Built with:
* Node.js v8.9
* Express
* Babel
* WebPack 4
* MongoDB

## Installation


### Requirements
* Node.js >= 8
* MongoDB >= 3.2
